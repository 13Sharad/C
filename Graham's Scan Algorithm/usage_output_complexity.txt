# Graham's Scan Algorithm - Sample Output, Usage, and Complexity

## Sample Output

Given the input points:
{ (0, 3), (2, 2), (1, 1), (2, 1), (3, 0), (0, 0), (3, 3) }

The program outputs the points on the convex hull in counterclockwise order: 

=> Points in the Convex Hull: (0, 3) (3, 3) (3, 0) (0, 0)


## Usage

1. Compile the code : gcc graham_scan.c -o graham_scan -lm

2. Run the program  : ./graham_scan


3. Modify the `points` array in `main()` to test different inputs.

## Complexity

- **Time Complexity**: `O(n log n)` due to sorting based on polar angle.
- **Space Complexity**: `O(n)` for the stack storing points on the convex hull.
